# 🛒 ShopNestApp

**ShopNestApp** is a full-stack e-commerce application where users can browse products, manage their cart, and securely make purchases using **Razorpay**. Built using **React (Vite)** on the frontend, **Spring Boot** on the backend, and **MySQL** for data storage.

---

## 🌟 Features

- 🔍 Browse a list of available products
- 🛒 Add products to the cart
- 💳 Secure payments via **Razorpay**
- ✅ Place and manage orders
- 👤 User authentication with role-based access (Admin / Customer)
- 🛠 Admin dashboard to add/edit/delete products
- 💾 Persistent data storage using MySQL

---

## 🧰 Tech Stack

| Layer     | Technology         |
|-----------|--------------------|
| Frontend  | React (Vite), HTML, CSS |
| Backend   | Spring Boot (Java) |
| Database  | MySQL              |
| Payments  | Razorpay API       |
| Auth      | JWT + Spring Security |
| Tools     | IntelliJ, MySQL Workbench, Postman |

---

## 💳 Razorpay Integration

- Customers can pay for products directly through Razorpay's secure payment gateway.
- After successful payment, the order details are saved and a confirmation is shown.
- The Razorpay API key and secret are securely managed in backend properties.
